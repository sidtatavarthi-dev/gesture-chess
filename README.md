Gesture-Controlled Chess

A chess game controlled by hand gestures using computer vision.

Features
- Hand tracking using MediaPipe
- Play chess by pointing and pinching
- Real-time camera feed display

Technologies
- Python
- OpenCV
- MediaPipe
- Pygame
- Python-chess

How to Run
1. Install dependencies: `pip install opencv-python mediapipe pygame python-chess numpy`
2. Run: `python GestureChess.py`
